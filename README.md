# Cinemarket ERC721 Wallet

## Installation

```
$ git clone git@github.com:CinemarketIO/wallet.git
$ npm i
$ npm run dev
```

## Run a production build with docker

```
$ docker build -t app .
$ docker run -d -p 80:80 app
```
